# 🛡️ Splunk SOC Detection Lab

## 📌 Overview

This project demonstrates hands-on experience with Splunk SIEM for a SOC Level 1 role.
It includes real-world detection use cases, SPL queries, dashboards, and incident investigations.

---

## 🎯 Objectives

* Develop detection use cases using Splunk SPL
* Simulate SOC monitoring scenarios
* Perform log analysis and threat detection
* Document investigations professionally

---

## 🧰 Tools Used

* Splunk Enterprise
* Sysmon
* Synthetic log dataset

---

## 🏗️ Lab Architecture

* Logs ingested into Splunk
* Data indexed in `practicelog`
* Detection queries executed via Search & Reporting

---

## 🔍 Detection Use Cases

This lab covers 25 SOC-relevant detection scenarios:

### 🔐 Authentication

* Brute Force Detection
* Login After Multiple Failures
* Privilege Escalation

### 🖥️ Endpoint Security

* PowerShell Abuse
* Suspicious Process Execution

### 🌐 Network Monitoring

* Port Scanning
* Suspicious Traffic

### 🧑‍💻 Insider Threat

* Lateral Movement
* Admin Misuse
